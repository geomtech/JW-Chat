Politique de confidentialité et de protection des données

Introduction

Bienvenue sur JW Chat (jw-ch.at). Votre confidentialité est importante pour nous. Cette page explique comment vos données sont collectées, utilisées et protégées.

Données Collectées

Pour créer un compte sur JW Chat, nous collectons uniquement les informations suivantes :

  • Adresse e-mail
  • Mot de passe

Utilisation des Données

Vos données sont utilisées dans les cas suivants :

  • Authentification et gestion de votre compte
  • Amélioration de nos services via l'API OpenAI
  • Communication en cas de mises à jour importantes

Partage des Données

Nous ne partageons pas vos données personnelles avec des tiers, sauf dans les cas suivants :

  • Conformité avec la loi ou les demandes des autorités légales
  • Fourniture des services via l'API OpenAI

Sécurité

Nous prenons des mesures appropriées pour protéger vos données, notamment :

  • Stockage sécurisé de vos mots de passe
  • Utilisation de connexions sécurisées (HTTPS)

Droits des Utilisateurs

Vous avez les droits suivants concernant vos données personnelles :

  • Accès à vos informations
  • Correction de vos données
  • Suppression de votre compte

Pour exercer ces droits, contactez-nous à l'adresse : support@jw-ch.at.
